# WhatsThat

## Overview

Third-year university module submission for **Mobile Application Development**. A WhatsApp clone built with **JavaScript, React Native, Node.js, and React.js**.

Designed to mimic the operation of a messaging application similar to WhatsApp. Users can send and receive messages, view chat histories and interact with contacts in real time.

## Installation / Setup

1. clone the repository
2. navigate into the project folder
3. 'npm install'
4. 'npx expo start'
5. 'w' for web

## Notes

The app connects to a RESTful API hosted externally by the module instructor. All API endpoints for messages, users and contacts are fully functional. No local server setup is required.
